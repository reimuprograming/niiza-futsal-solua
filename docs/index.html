フットサルチームSOLUAの紹介

フットサルチームSOLUAは、経験者と初心者関係なくとにかく楽しく上を目指す！という方針で活動しています。

現役県リーガーの丁寧な指導のもとで練習しているので、フットサルの戦術を基礎からしっかり学びたい人にとって最適な環境です。

なんと、チームメンバーの中からスポンサーとなってチームの活動を応援して下さる人もいます。チームの練習着・移動着、ボールはすべてスポンサーが支給してくださいます。また、練習施設の提供もして頂いています。

メンバーは常に募集しています！現在のメンバーの多くは初心者から始めているので、初心者の人も大歓迎です。

興味のある方は先ずゲストとして個人参加して頂き、チームの雰囲気を感じて頂ければ幸いです。

メンバー構成

現在、メンバーは１８人！

メンバーは10代から50代、未経験から始めた人や女性、外国出身の人もいます。元県リーガーやフットサル経験の長い人もいるので初心者へのフォローアップの環境も整っています。

一人ひとり意識高く持っていて、その上で楽しい雰囲気で活動しています。

活動について

練習は基本的に週２回19:00~22:00で活動しています。

土曜リーグ18:00~22:00、日曜リーグ12:00~16:00もそれぞれ月１回ぐらいのペースで参加しています。（2018年度上期日曜リーグ優勝）

活動場所

  • 新座市民総合体育館（本拠点）
  • 所沢特別支援学校体育館（第２拠点）
